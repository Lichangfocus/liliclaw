# 目标与计划（PLAN · OKR 体系）

> 基于北极星逐层拆解的计划框架：使命 → 月 → 周，越往下越具体、越可量化。
> 周 KR 最细、带进度，是我每天对照检查的东西。初始化运行搭出第一版，每周复盘班更新，工作台「目标与计划」直接渲染下方 JSON。

```json
{
  "updated": "",
  "mission": "{{GOAL}}",
  "cycles": []
}
```

（cycle 结构：{"period":"本周 2026-Wxx"|"本月 2026-xx","state":"进行中","objectives":[{"name":"O1 …","krs":[{"t":"KR 描述","n":"当前/目标","pct":0}]}]}）
